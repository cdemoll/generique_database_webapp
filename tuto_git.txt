Consigne d'utilisation de git sous Linux

Pour importer le projet de GitHub à votre répertoire en local:
> cd /votre_repertoire
> git init
> git clone https://github.com/Ephemeral25/generique_database_webapp.git
> git add .

Le projet est maintenant sur votre machine et sous l'environnement git

Pour que chacun puisse travailler de manière organisée il est nécessaire d'utiliser les branches, pour ce faire:
> git branch <votre_branche>  // créer une branche
> git checkout <votre_branche> // vous met sur votre branche
> git status //retourne l'emplacement sur lequel on est
